TSTT Software

Information related to accessing, downloading and using the TSTT center software will reside on this page.

Pointers to existing center software

  • FMDB (RPI): Mesh management infrastructure
  • CUBIT (SNL): Unstructured hex and tet cad-based mesh generation
  • Frontier (BNL/SB): Front tracking for surfaces of discontinuity
  • Mesquite (SNL/ANL): Mesh quality improvement
  • NWGrid (PNL): Mesh generation and adaptive mesh management
  • Opt-MS (ANL): Mesh quality improvement
  • Overture (LLNL): an object-oriented framework for solving PDEs in complex geometries
  • Trellis (RPI): C++ software framework for geometry-based, numerical mechanics applications

The interface definition effort.

  • The base interface specification is underway and the latest draft [pdf] is available.
  • The geometry interface specification is underway and the latest draft [pdf] is available.
  • The mesh interface specification is underway and the latest draft [pdf] is available.

    • This specification is relatively new and always evolving. Therefore, it is not guaranteed to be either complete or stable at this time, but serves as an indication of our current thoughts. Changes to this site will likely occur every two to three weeks so check back regularly!
