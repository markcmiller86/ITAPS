Detailed Description

Virtual base class for 2D and 3D swapping classes.

The swapping classes hold minimal state data (an iMesh instance and some control options); they're mostly a way to encapsulate algorithms within a common interface.

iBase_TagHandle *  ptr_tag_handle = NULL 
) throw (iBase_ErrorType) [pure virtual]

Swap the entire mesh or entity set.

Checks all entities in the given set for swapping. Multiple passes are made through the mesh to ensure that all candidate swaps are considered; a second call to this function with the same swapping parameters should produce no additional change to the mesh.

An optional tag handle argument allows fine-grained control: a d-1 dimensional entity is considered for swapping only if the integer tag value is non-zero; if the tag is not set, a value of zero (no swapping) is inferred.

Parameters:
[in] entity_set The set whose entities will be considered for swapping; passing the root set specifies swapping for the entire mesh.
[in] ptr_tag_handle An optional pointer to an integer-valued tag. If this tag handle is provided, only d-1 dimensional entities with a tag value set != 0 will be checked for swapping.
Returns:
Total number of swaps performed.

Implemented in ITAPS_Swap::Swap2D, and ITAPS_Swap::Swap3D.

bool ITAPS_Swap::Swap::qCreateFromVerts [protected]

Create triangles and tets directly from verts if this is true.

Some implementations support creation of triangles and tetrahedra only from vertices, others only from edges (2D) or faces (3D), still others from both. This flag tells the swapping code which to use.
