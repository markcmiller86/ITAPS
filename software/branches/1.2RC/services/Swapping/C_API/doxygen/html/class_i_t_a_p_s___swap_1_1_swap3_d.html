Detailed Description

A specialization of Swap for 3D meshes.

Because of the possibility of changes to the boundary during 3D swapping, some additional publicly available functionality is defined to control changes to the boundary.

Constructor & Destructor Documentation

ITAPS_Swap::Swap3D::Swap3D (  )  [inline]

Default constructor.

By default, edge swapping is enabled (in addition to face swapping). Also by default, pairs of boundary faces that are nearly coplanar and classified on the same geometric face can be reconfigured.
