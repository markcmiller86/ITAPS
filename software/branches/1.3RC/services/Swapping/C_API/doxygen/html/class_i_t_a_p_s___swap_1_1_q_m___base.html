Detailed Description

A virtual base class providing a common interface to all quality measures.

Important caveat: quality measures must be cell-based. That is, the calculation must be based solely on the coordinates of the vertices of a tetrahedron. While there are other useful quality measures, this restricted form is what the swapping drivers expect.
